Cambiar la ruta de la carpeta personal (en el ClienteCloud) que se quiere sincronizar y la ruta de la carpeta 
que actuará como servidor (en el ServidorCloud) donde se almacenarán los ficheros sincronizados.
También cambiar el atributo username del ClienteCloud si se quiere probar con un usuario distinto al predeterminado

En el package "comun"  existe una clase Main donde modificando los datos puedes crear un nuevo usuario que se 
almacenará en el XML Users.xml
